# IRGen Implementation Plan (path 1: minimal self-resolve)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `IRGen` — translate `CompUnit` AST into linear non-SSA IR (`alloca/load/store`), wire `-dump-ir` end-to-end, validated by golden-file regression tests.

**Architecture:** Single `IRGenerator` class extending `ASTVisitor` (path 1 = interface doc §3 fallback). Symbol stack + constant folder + loop stack are internal members of `IRGenerator` (not a separate `src/sema/` — temporary, swapped out when real Sema lands). Each local/param gets an `alloca` slot in the `entry` block; reads/writes become `load`/`store`. Globals registered via `Module::create_global`; const refs inlined as immediates.

**Tech Stack:** C++20, CMake 3.20+, CTest. No third-party libs. Tests = end-to-end golden-file (`-dump-ir` output diffed against `.expected`), mirroring the parser regression harness.

**Authority:** Spec [`docs/superpowers/specs/2026-06-24-irgen-design.md`](../specs/2026-06-24-irgen-design.md).

## Global Constraints

- C++20, `-Wall -Wextra -Wpedantic` (already in `CMakeLists.txt`).
- Headers: `#pragma once`, `#include "toyc/xxx.h"` for project headers, `<std>` for stdlib, `namespace toyc { ... }  // namespace toyc`.
- snake_case names; `enum class` with PascalCase enumerators.
- Fail fast: assume input already passed `validate_comp_unit`; on unreachable states emit a `DiagnosticEngine::error(DiagnosticStage::Sema, ...)` and return failure. No defensive fallbacks.
- IR text format is frozen — see `src/ir/ir_printer.cpp`. Golden `.expected` files must match it byte-for-byte (trailing newline included). `%v.N` ids come from `Module::fresh_id()`, monotonically increasing in emission order.
- `alloca` for locals/params is inserted at the **front** of the `entry` block (via `entry->push_front`), independent of the current insert point, so all allocas cluster at the entry top (prep for mem2reg).
- All commands assume build dir `build/`. Reconfigure if needed: `cmake -S . -B build`.

`include/toyc/irgen.h`:
```cpp
#pragma once

#include "toyc/ast.h"
#include "toyc/ast_contract.h"
#include "toyc/ast_visitor.h"
#include "toyc/ir.h"

#include <memory>
#include <optional>
#include <string>
#include <unordered_map>
#include <vector>

namespace toyc {

class DiagnosticEngine;

struct Symbol {
    SymbolStorageKind kind;
    Value* addr = nullptr;                 // alloca slot (local var/param) or global addr
    std::optional<int> const_value;        // for const (local or global)
};

class IRGenerator : public ASTVisitor {
public:
    std::unique_ptr<Module> generate(const CompUnit& unit, DiagnosticEngine& diag);

    // CompUnit / globals / funcs
    void visit_comp_unit(const CompUnit& unit) override;
    void visit_global_const(const GlobalConstDecl& decl) override;
    void visit_global_var(const GlobalVarDecl& decl) override;
    void visit_func_def(const FuncDef& func) override;

    // stmts
    void visit_block_stmt(const BlockStmt& node) override;
    void visit_empty_stmt(const EmptyStmt& node) override;
    void visit_expr_stmt(const ExprStmt& node) override;
    void visit_assign_stmt(const AssignStmt& node) override;
    void visit_const_decl_stmt(const ConstDeclStmt& node) override;
    void visit_var_decl_stmt(const VarDeclStmt& node) override;
    void visit_if_stmt(const IfStmt& node) override;
    void visit_while_stmt(const WhileStmt& node) override;
    void visit_break_stmt(const BreakStmt& node) override;
    void visit_continue_stmt(const ContinueStmt& node) override;
    void visit_return_stmt(const ReturnStmt& node) override;

    // exprs (return the produced Value* via last_value_)
    void visit_int_literal(const IntLiteralExpr& node) override;
    void visit_ident(const IdentExpr& node) override;
    void visit_binary(const BinaryExpr& node) override;
    void visit_unary(const UnaryExpr& node) override;
    void visit_call(const CallExpr& node) override;

private:
    // Expression evaluation: walks expr and leaves result in last_value_.
    Value* eval_expr(const Expr& expr);
    // Constant folding; nullopt if not foldable.
    std::optional<int> try_fold(const Expr& expr) const;

    // Symbol resolution.
    void push_scope();
    void pop_scope();
    void declare(const std::string& name, Symbol sym);
    Symbol* resolve(const std::string& name);

    // Helpers.
    Value* alloca_in_entry();               // alloca pushed to entry front
    BasicBlock* current_block() const { return builder_->insert_point(); }

    DiagnosticEngine* diag_ = nullptr;
    std::unique_ptr<Module> module_;
    std::unique_ptr<IRBuilder> builder_;
    BasicBlock* entry_ = nullptr;           // current function's entry block
    FuncReturnType current_ret_ = FuncReturnType::Int;
    FuncSignatureMap func_sigs_;

    // Loop context for break/continue.
    struct LoopFrame { BasicBlock* cond; BasicBlock* exit; };
    std::vector<LoopFrame> loops_;

    // Scope stack + expression result hand-off.
    std::vector<std::unordered_map<std::string, Symbol>> scopes_;
    Value* last_value_ = nullptr;
    bool had_error_ = false;
};

std::unique_ptr<Module> generate(const CompUnit& unit, DiagnosticEngine& diag);

}  // namespace toyc
```

`src/ir/irgen.cpp` (Task 1 subset — globals scan + func skeleton + return-const only; other visits added in later tasks):
```cpp
#include "toyc/irgen.h"

#include "toyc/ast_access.h"
#include "toyc/diagnostics.h"
#include "toyc/ir_builder.h"

#include <memory>

namespace toyc {

namespace {

FuncRet to_func_ret(FuncReturnType t) {
    return t == FuncReturnType::Int ? FuncRet::Int : FuncRet::Void;
}

}  // namespace

Value* IRGenerator::alloca_in_entry() {
    auto inst = std::make_unique<AllocaInst>(module_->fresh_id());
    Value* raw = inst.get();
    entry_->push_front(std::move(inst));
    return raw;
}

void IRGenerator::push_scope() { scopes_.emplace_back(); }
void IRGenerator::pop_scope() { scopes_.pop_back(); }

void IRGenerator::declare(const std::string& name, Symbol sym) {
    scopes_.back().emplace(name, std::move(sym));
}

Symbol* IRGenerator::resolve(const std::string& name) {
    for (auto it = scopes_.rbegin(); it != scopes_.rend(); ++it) {
        auto found = it->find(name);
        if (found != it->end()) {
            return &found->second;
        }
    }
    return nullptr;
}

std::optional<int> IRGenerator::try_fold(const Expr& expr) const {
    // Implemented in Task 4.
    (void)expr;
    return std::nullopt;
}

Value* IRGenerator::eval_expr(const Expr& expr) {
    last_value_ = nullptr;
    walk_expr(expr, *this);
    return last_value_;
}

void IRGenerator::visit_global_const(const GlobalConstDecl& decl) {
    auto folded = try_fold(*decl.init);
    if (!folded) { had_error_ = true; return; }
    module_->create_global(decl.name, *folded, /*is_const=*/true);
    declare(decl.name, Symbol{SymbolStorageKind::GlobalConst, nullptr, *folded});
}

void IRGenerator::visit_global_var(const GlobalVarDecl& decl) {
    auto folded = try_fold(*decl.init);
    if (!folded) { had_error_ = true; return; }
    GlobalVar* gv = module_->create_global(decl.name, *folded, /*is_const=*/false);
    declare(decl.name, Symbol{SymbolStorageKind::GlobalVar, gv->addr, std::nullopt});
}

void IRGenerator::visit_comp_unit(const CompUnit& unit) {
    func_sigs_ = build_func_signature_map(unit);
    push_scope();  // global scope
    for (const CompUnit::Item& item : unit.items) {
        if (item.kind == CompUnit::ItemKind::GlobalConst) {
            visit_global_const(item.global_const);
        } else if (item.kind == CompUnit::ItemKind::GlobalVar) {
            visit_global_var(item.global_var);
        } else {
            visit_func_def(item.func_def);
        }
    }
    pop_scope();
}

void IRGenerator::visit_func_def(const FuncDef& func) {
    Function* fn = module_->create_function(func.name, to_func_ret(func.return_type),
                                            static_cast<unsigned>(func.params.size()));
    entry_ = fn->create_block();
    builder_ = std::make_unique<IRBuilder>(*module_, entry_);
    current_ret_ = func.return_type;
    push_scope();  // function scope (params + top-level locals), sits on the global scope

    // Params: alloca slot + store %arg.N.
    for (unsigned i = 0; i < func.params.size(); ++i) {
        Value* slot = alloca_in_entry();
        builder_->create_store(slot, fn->param(i));
        declare(func.params[i].name, Symbol{SymbolStorageKind::Param, slot, std::nullopt});
    }

    walk_block(*func.body, *this);

    pop_scope();
    entry_ = nullptr;
}

// --- Stubs added/overridden in later tasks; minimal bodies here keep it compiling. ---
void IRGenerator::visit_block_stmt(const BlockStmt& node) { push_scope(); walk_block(node, *this); pop_scope(); }
void IRGenerator::visit_empty_stmt(const EmptyStmt&) {}
void IRGenerator::visit_expr_stmt(const ExprStmt& node) { eval_expr(*node.expr); }
void IRGenerator::visit_assign_stmt(const AssignStmt&) {}
void IRGenerator::visit_const_decl_stmt(const ConstDeclStmt&) {}
void IRGenerator::visit_var_decl_stmt(const VarDeclStmt&) {}
void IRGenerator::visit_if_stmt(const IfStmt&) {}
void IRGenerator::visit_while_stmt(const WhileStmt&) {}
void IRGenerator::visit_break_stmt(const BreakStmt&) {}
void IRGenerator::visit_continue_stmt(const ContinueStmt&) {}
void IRGenerator::visit_return_stmt(const ReturnStmt& node) {
    Value* v = nullptr;
    if (node.value && node.value->has_value()) {
        v = eval_expr(**node.value);
    }
    builder_->create_ret(v);
}
void IRGenerator::visit_int_literal(const IntLiteralExpr& node) {
    last_value_ = module_->get_constant(node.value);
}
void IRGenerator::visit_ident(const IdentExpr&) {}
void IRGenerator::visit_binary(const BinaryExpr&) {}
void IRGenerator::visit_unary(const UnaryExpr&) {}
void IRGenerator::visit_call(const CallExpr&) {}

std::unique_ptr<Module> IRGenerator::generate(const CompUnit& unit, DiagnosticEngine& diag) {
    diag_ = &diag;
    module_ = std::make_unique<Module>();
    visit_comp_unit(unit);
    if (had_error_) {
        diag.error(DiagnosticStage::Sema, SourceLoc{0, 0}, "IRGen failed (unfoldable constant initializer)");
        return nullptr;
    }
    return std::move(module_);
}

std::unique_ptr<Module> generate(const CompUnit& unit, DiagnosticEngine& diag) {
    IRGenerator g;
    return g.generate(unit, diag);
}

}  // namespace toyc

void IRGenerator::visit_binary(const BinaryExpr& node) {
    Value* lhs = eval_expr(*node.lhs);
    Value* rhs = eval_expr(*node.rhs);
    switch (node.op) {
        case BinaryOp::Add: last_value_ = builder_->create_binary(Opcode::Add, lhs, rhs); break;
        case BinaryOp::Sub: last_value_ = builder_->create_binary(Opcode::Sub, lhs, rhs); break;
        case BinaryOp::Mul: last_value_ = builder_->create_binary(Opcode::Mul, lhs, rhs); break;
        case BinaryOp::Div: last_value_ = builder_->create_binary(Opcode::Sdiv, lhs, rhs); break;
        case BinaryOp::Mod: last_value_ = builder_->create_binary(Opcode::Srem, lhs, rhs); break;
        case BinaryOp::Lt: last_value_ = builder_->create_icmp(Opcode::ICmpSlt, lhs, rhs); break;
        case BinaryOp::Le: last_value_ = builder_->create_icmp(Opcode::ICmpSle, lhs, rhs); break;
        case BinaryOp::Gt: last_value_ = builder_->create_icmp(Opcode::ICmpSgt, lhs, rhs); break;
        case BinaryOp::Ge: last_value_ = builder_->create_icmp(Opcode::ICmpSge, lhs, rhs); break;
        case BinaryOp::Eq: last_value_ = builder_->create_icmp(Opcode::ICmpEq, lhs, rhs); break;
        case BinaryOp::Ne: last_value_ = builder_->create_icmp(Opcode::ICmpNe, lhs, rhs); break;
        case BinaryOp::And:  // short-circuit added in Task 7
        case BinaryOp::Or:
            had_error_ = true; break;
    }
}

void IRGenerator::visit_unary(const UnaryExpr& node) {
    Value* operand = eval_expr(*node.operand);
    switch (node.op) {
        case UnaryOp::Plus: last_value_ = operand; break;
        case UnaryOp::Minus: last_value_ = builder_->create_neg(operand); break;
        case UnaryOp::Not: {
            Value* zero = module_->get_constant(0);
            last_value_ = builder_->create_icmp(Opcode::ICmpEq, operand, zero);
            break;
        }
    }
}
```

std::optional<int> IRGenerator::try_fold(const Expr& expr) const {
    if (const IntLiteralExpr* lit = as_int_literal(expr)) {
        return lit->value;
    }
    if (const IdentExpr* id = as_ident(expr)) {
        for (auto it = scopes_.rbegin(); it != scopes_.rend(); ++it) {
            auto found = it->find(id->name);
            if (found != it->end() && found->second.const_value) {
                return found->second.const_value;
            }
        }
        return std::nullopt;
    }
    if (const BinaryExpr* b = as_binary(expr)) {
        auto l = try_fold(*b->lhs);
        auto r = try_fold(*b->rhs);
        if (!l || !r) return std::nullopt;
        switch (b->op) {
            case BinaryOp::Add: return *l + *r;
            case BinaryOp::Sub: return *l - *r;
            case BinaryOp::Mul: return *l * *r;
            case BinaryOp::Div: return *r == 0 ? std::optional<int>{} : *l / *r;
            case BinaryOp::Mod: return *r == 0 ? std::optional<int>{} : *l % *r;
            case BinaryOp::Lt: return *l < *r;
            case BinaryOp::Le: return *l <= *r;
            case BinaryOp::Gt: return *l > *r;
            case BinaryOp::Ge: return *l >= *r;
            case BinaryOp::Eq: return *l == *r;
            case BinaryOp::Ne: return *l != *r;
            case BinaryOp::And: return *l && *r;
            case BinaryOp::Or: return *l || *r;
        }
        return std::nullopt;
    }
    if (const UnaryExpr* u = as_unary(expr)) {
        auto v = try_fold(*u->operand);
        if (!v) return std::nullopt;
        switch (u->op) {
            case UnaryOp::Plus: return *v;
            case UnaryOp::Minus: return -*v;
            case UnaryOp::Not: return !*v;
        }
    }
    return std::nullopt;
}

    Value* slot = alloca_in_entry();
    Value* zero = module_->get_constant(0);
    Value* one = module_->get_constant(1);

    Value* lhs = eval_expr(lhs_expr);
    Value* lhs_bool = builder_->create_icmp(Opcode::ICmpNe, lhs, zero);

    BasicBlock* rhs_bb = entry_->parent()->create_block();
    BasicBlock* short_bb = entry_->parent()->create_block();
    BasicBlock* merge_bb = entry_->parent()->create_block();
    // is_and: lhs false -> short-circuit to short_bb (store 0); else eval rhs.
    // (||):   lhs true  -> short_bb (store 1); else eval rhs.
    builder_->create_cond_br(lhs_bool, is_and ? rhs_bb : short_bb,
                                      is_and ? short_bb : rhs_bb);

    builder_->set_insert_point(short_bb);
    builder_->create_store(slot, is_and ? zero : one);
    builder_->create_br(merge_bb);

    builder_->set_insert_point(rhs_bb);
    Value* rhs = eval_expr(rhs_expr);
    Value* rhs_bool = builder_->create_icmp(Opcode::ICmpNe, rhs, zero);
    builder_->create_store(slot, rhs_bool);
    builder_->create_br(merge_bb);

    builder_->set_insert_point(merge_bb);
    return builder_->create_load(slot);
}
